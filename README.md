# main-web-guide

## 项目简介

`main-web-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用 HTML 页面集合的存储与展示空间。每个页面均可独立访问，适合用于快速分享静态内容或作为临时展示页面。

## 目录说明

```
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源目录（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要存放各独立 HTML 页面文件，每个文件应自包含或引用 `assets/` 中的资源。
- `assets/`：存放页面公用的样式、脚本或图片文件，便于统一管理。
- `index.html`：可选，可作为仓库的导航或介绍页。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面可单独通过其文件路径访问。页面内容可能包含信息展示、工具界面、实验性功能等，不限定使用场景。新增页面时，请遵循以下基本规则：

- 页面文件放置在 `pages/` 目录下，命名清晰且避免特殊字符。
- 如需使用外部资源，优先引用 `assets/` 中的本地文件。
- 页面之间应保持独立，避免相互依赖。

## 维护说明

- 欢迎提交新的独立 HTML 页面，请确保内容合法合规且不包含恶意代码。
- 提交前请检查页面在主流浏览器中的基本兼容性。
- 不建议在页面中引用外部不可控资源（如第三方 CDN），以免因外部变更导致页面失效。
- 仓库维护者会定期清理失效或不再适用的页面，保持归档内容的质量。

本仓库仅作为静态 HTML 页面的归档与分发渠道，不对页面内容的准确性、时效性或适用性作任何保证。使用者应自行判断页面内容的适用性。
